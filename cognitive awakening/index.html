# 每日反思

描述经过、询问原因、改进措施

小事听从脑，大事听从心

# 捕捉感性、潜意识：

“最” 触动自己的点
“总” 不自觉跳出来的重复念头，挥之不去的事
无意识的第一反应
梦境
身体的不适
直觉

# 元认知

把所有想做的事情 都列出来，进行排序，找出最重要的事，让大脑清醒。
### 掌控思维
针对当下的时间，保持觉知，审视第一反应，产生明确的主张；
针对全天的日程，保持清醒，时刻明确下一步要做的事情；
针对长远的目标，保持思考，想清楚长远意义和内在动机。

# 正确的方法” 四个特征：

第一，有明确的目标。
第二，练习时极度专注。事前聚精会神，事后完全忘记。
第三，能获得有效的反馈。
第四，始终在拉伸区练习。

# 专注于深度学习，同时对浅学习保持开放。

# 关联和自己有关的。

# 获取关联能力

首先，手中有锤子。其他事物对自己有意义，是钉子。
再次，保持好奇心。
最后，常说一句话。只要触动到了自己，就要想办法让它效率最大化，也就是主动关联到别处，并让自己的行动发生改变。“这个道理还能用在什么地方？”

# 建立个人认知体系

只有当知识能够帮助你做实际决策的时候，它才是我的知识。

# 只学让自己触动的

用自己的话把所学表达出来

将别人的意见与情绪分离，单独处理，意见要好好思考，情绪不用理会，不过如果想分析他人对自己的看法可以考虑一下。

我觉得挺好的啊，为什么这么多人踩，前面确实很多科学知识，缝合了各种书籍思想，
我不知道他所引用的是不是都是自己创造的观点，我也都没看过，但是很多思想相通。
下一本想看《别逗了，费曼先生》
是不是小说受众最大啊，没加过哪本小说被喷的厉害，但是有些小说我能汲取到的点太少了，什么时候会去看历史小说哈哈，以后到了这个时候来反馈哈哈，希望会有这么一天。

要好好培养自己的习惯，每日反思
